The Test Frame 1.0 Manual
Welcome to the Test Frame 1.0 Manual. Test Frame is a framework designed to simplify the efforts codec developers has spent on finding out to which extent the developed codec has been qualified according to current standards and thus to improve the efficiency of CI procedure when a codec is being developed. This Manual provides users simple guides how the framework could be used.
