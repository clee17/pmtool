Accounts
On this page
In order to use CIDANA Test Framework, a user needs to own an account first. Every test would be run by an individual who is registered with Test Framework. The testing results shall be limited to registered users only.
Although there is indeed an option that the Test Framework is completely deployed locally, the login system is still kept due to the reason that the results and analyze reports could be managed and stored more safely.
CIDANA Test Framework has provided some initial accounts for a user to start with, yet users are more than welcome to register and set up their own accounts.
Default accounts
As soon as the Test Framework is deployed for the first time, you could use below accounts to login into the system.
Usernamepassword
Guest01123
However, it is highly recommended to set up your own account as soon as possible.
Add an account
1. Add an account through management portal
An account could be added through the management panel where an administrator could easily add, delete and manage the accounts.
However, please kindly be aware that the management panel is only available in higher version (Test Framework 1.4.2).
2. Add an account through CIDANA
For those who are using versions lower than 2.4.0, the users and maintenance will be completed by the CIDANA support team, please kindly contact your person of contact in CIDANA to update your accounts.
