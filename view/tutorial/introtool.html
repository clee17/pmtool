PM使用的工具介绍

本篇对本公司使用的几样PM工具以及常用的Excel表格进行简单的介绍，

具体工具使用包括以下几个类别：

  • 时间管理类
  • 文件传输类
  • 记录汇报类
  • 沟通交流类

时间管理类

 Microsoft To Do 

简单的个人时间管理软件，类似的产品有很多，选择自己喜欢的使用即可。

循环开设的任务主要如下：

此外有些项目会有规律性的类似于汇报会议等内容，可以根据需要自行添加。

项目托管类

 Github Account 

在github上我们存放了部分可以向大众公开的项目，与此同时我们也使用github做一些项目管理的Task Manager工作。

地址：+Cidana Task Manager+

github可以使用自己的账号也可以新建账号，要求IT相关人员将你的账号加入即可使用。

 Cidana Git 

Cidana git是在我们公司内部服务器上搭建起来的项目托管仓库，一些处于开发阶段的代码会放在上面，PM的assets库也保存在其上。

地址：+Cidana git+

该仓库使用Cidana账号登陆即可，需要联系IT给予授权。

 File Zilla 

公司使用的FTP传输工具，主要用于对客户发布和传输必备的镜像文件、sdk文件等等内容使用。

基本保持一个客户一个账号的原则，如果该客户在客户信息中没有开设FTP账户，或已开设的FTP账户失效，需要联系IT人员进行重新分配。

可依照自己的喜好使用自己习惯使用的FTP传输工具。
